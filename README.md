# Список исправленных ошибок в соответствии со стандартом Epic Games C++ Coding Style
## Вариант 1
### Task 1:
1. Глобальные директивы
  - Убрано using namespace std в глобальной области видимости, добавлен префикс std:: ко всем объектам стандартной библиотеки
2. Именование функций
  - Функции переименованы в PascalCase (с заглавной буквы)
  - Убраны нижние подчёркивания из имён функций
  - Пример: find_numbers → FindNumbers
3. Именование параметров функций
  - Параметры переименованы в PascalCase (с заглавной буквы)
  - Имена параметров сделаны информативными (отражают их назначение)
  - Пример: Mvector → Matrix, i → Row, j → Col, Wvector → Weights
4. Именование переменных в main()
  - Все переменные переименованы в PascalCase (с заглавной буквы)
  - Пример: weight_vector → Weights, main_vector → WeightMatrix
5. Инициализация переменных
  - Все переменные инициализированы при объявлении (присвоены начальные значения)
  - Пример: int N = 0, B = 0, Weight = 0, Volume = 0, IndexI = 0, IndexJ = 0;
6. Форматирование кода
  - Добавлены фигурные скобки для всех управляющих конструкций (if, for, while) даже с одной строкой
  - Фигурные скобки открываются в новой строке
  - Добавлены пробелы вокруг бинарных операторов ('+', '-', '*', '/', '=')
  - Убраны лишние пробелы в угловых скобках: vector <vector <int>> → std::vector<std::vector<int>>
7. Инкремент/декремент
  - Заменён постфиксный инкремент/декремент на префиксный
  - Пример: i++ → ++i
8. Логика алгоритма
  - Добавлена матрица CountMatrix для отслеживания количества предметов
  - Исправлена логика выбора: сначала максимизируется количество предметов, затем близость к B
  - Исправлена функция восстановления пути: добавлено вычитание веса текущего предмета
  - Улучшен поиск оптимального решения (перебор по всем возможным весам)
9. Работа с файлами
  - Добавлена проверка открытия файла
  - Добавлены проверки корректности вводимых данных
  - Добавлено закрытие файла после использования
  - Добавлены вспомогательные функции: OpenFile, ReadNumber, ReadArray
10. Локализация
  - Добавлена поддержка русского языка: setlocale(LC_ALL, "Russian");
11. Структура проекта
  - Программа разделена на модули согласно стандарту Epic Games
  - Создан заголовочный файл Task1.h с прототипами функций
  - Реализация функций вынесена в отдельный файл Task1cpp.cpp
  - В основном файле Task1.cpp оставлена только функция main
